## Overview

`labs.eknathalabs.com` is the hands-on learning subdomain of [eknathalabs.com](https://eknathalabs.com), focused on real terminal labs, production-grade scenarios, and interactive playgrounds for DevOps and Platform Engineering.

The site is currently **under active construction**. The homepage serves as a landing page with early access sign-up while content is being built out.

---

## Tech Stack

- Pure HTML / CSS / JavaScript — no frameworks, no build step
- Fonts: [Sora](https://fonts.google.com/specimen/Sora) + [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) via Google Fonts
- Hosted via static hosting (GitHub Pages / Cloudflare Pages / any static host)

## Deployment

This site is designed for any static host. Recommended options:

- **GitHub Pages** — push to `main` branch, enable Pages in repo settings
- **Cloudflare Pages** — connect repo, auto-deploy on push
- **Netlify** — drag and drop the folder or connect via Git
